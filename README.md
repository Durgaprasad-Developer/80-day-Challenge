# 🔥 80 Days of Discipline

> **This is a personal progress tracker, not a product.**
> Everything here is vibe-coded. I just wanted something to track my streak every day. Simple as that.

---

## What is this?

A minimal, offline-first discipline tracker I built to hold myself accountable for **80 days** of consistent daily practice.

No backend. No database. No accounts. Just me, my browser, and `localStorage`.

---

## What I'm tracking

- **Reinforcement Learning** — Daily RL study/practice
- **Pushups** — Progressive reps from 21 → 100 over 80 days

---

## Features

- ✅ Daily checklist (RL + Pushups)
- 🔥 Streak counter (breaks if you miss a day)
- 📊 XP system + Level progression
- 🟩 GitHub-style 80-day heatmap with hover tooltips
- 🏅 Milestone badges (Day 10, 20, 30, 50, 80)
- 💾 JSON backup/restore (1-click export)
- 📱 PWA — installable on mobile, works offline

---

## Stack

- Pure HTML + Vanilla JS + CSS
- Zero frameworks, zero dependencies
- LocalStorage for persistence
- Service Worker for offline support
